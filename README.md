# AutoSimX

## Virtual AUTOSAR ECU Communication Simulator

AutoSimX is a software-only simulation platform that demonstrates AUTOSAR-inspired ECU communication using a virtual CAN bus.

## Features

- Virtual CAN Bus
- ECU Simulation
- AUTOSAR-inspired Layered Architecture
- Live CAN Trace
- Message Logging
- Fault Injection
- Vehicle Dashboard

## Technologies

- Python
- PySide6
- Object-Oriented Programming
- Git
- GitHub
